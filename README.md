# CharlesLoader
Charles v4.x 动态启动loader

#####使用方法
* Windows系统下将所有文件放置在Charles/lib/下运行bat即可，Linux和osx可以使用自行看bat中的命令启动

######此方法无需破解jar包即可完成charles的破解，并且不影响升级
